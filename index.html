iOS Application

Iris Lens

Project Manager UI/UX Design Icon Design Accessibility User Research

My Role & Focus

In the development of this application, I took on the dual responsibility of Project Manager and Designer. My core mission was to bridge user needs with accessible tech solutions, ensuring that individuals with color vision deficiencies could interact with the app naturally.

  • • User Research & Insights: I conducted user interviews with target users experiencing different forms of color blindness (Protanomaly, Deuteranomaly, Tritanomaly) to pinpoint their real-world navigation challenges.
  • • UI/UX & Accessibility: Designed the high-contrast interface and customized real-time camera views, focusing on spatial layout, clear color readouts, and accessible visual assets suited for audiences with color vision deficiencies.
  • • App Icon Design: Created the official application icon, ensuring the visual metaphor reflects clarity, color accessibility, and Apple App Store design guidelines.

AmazeD: Wizard's Adventure

Project Manager Game Design UI/UX Design 3D Modeling Ideation

My Role & Focus

For this cross-device multiplayer project, I acted as the Project Manager, lead Game Designer, and 3D Modeler. I conceptualized the game's core loop, designed its multi-device UI/UX workflow, and created all visual assets and characters.

  • • Ideation & Concept: Co-created the game's original concept, pitching a local party setup that uses the iPhone's gyroscope as a steering input device.
  • • Game Design: Designed all gameplay environments, levels, labyrinth layouts, buttons, and custom layout graphics.
  • • UI/UX Design: Created the user experience interfaces for both the main display (macOS/tvOS splitscreen) and the iPhone mobile controller screen.
  • • 3D Modeling & Characters: Designed and developed the 3D models for all in-game assets, labyrinth walls, items, and the playable wizard characters.
